Rewinder Accuracy Root-Cause Investigation

Challenge: Production rewinders were reporting inconsistent tape lengths, affecting downstream measurement confidence and process reliability.

Engineering: Verified PLC distance logic, encoder assumptions, and machine behavior before isolating the true physical source of systematic length error.

Result: Produced the technical evidence that supported supplier-side corrective action and improved confidence in rewinder accuracy analysis.

Inline Width Traceability and Operator Monitoring System

Challenge: Production needed a reliable way to capture width data, relate it to machine motion, and make it useful during operation and after the run.

Engineering: Moved from early PLC-side logging investigation to a final EtherNet/IP-based architecture with a reconnect-aware Windows Service and tray-based operator inspection workflow.

Result: Built a structured traceability system with live monitoring, run-based logging, metadata, and completed-run inspection tools.
